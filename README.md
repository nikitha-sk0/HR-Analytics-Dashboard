# HR Analytics Dashboard

## Project Title
HR Analytics Dashboard for Employee Attrition Analysis

## Objective
The objective of this project is to analyze employee attrition patterns using Power BI and identify key factors affecting employee turnover.

## Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Data Visualization

## Dashboard Insights
The dashboard analyzes employee attrition based on:
- Age group
- Salary range
- Job role
- Education
- Years at company
